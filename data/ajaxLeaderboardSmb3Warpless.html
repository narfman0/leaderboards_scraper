  1. Anything not possible on an original and unmodified NES, brick controller, and SMB3 cartridge is prohibited for runs. This includes but is not limited to: Turbo, left+right, savestates, using a non-official rom, speeding up gameplay, rewind (such as on the Switch virtual console).

  2. When starting a new run, you must go to the title screen through means of a soft or hard reset. This means that NG+ type runs are not allowed, which includes starting a run on the title screen with RAM storage or an inventory full of p-wings after previously beating the game + pressing start after credits.

  3. Run highlights should be posted to YouTube or Twitch, videos from any other site will most likely be rejected. Make sure that the video won't automatically expire (e.g. a twitch broadcast) and create permanent highlights instead.

  4. The full run must be seen in the submission video (beginning and start points can't be cut off whatsoever). It is encouraged that the moments before your run are also included, such as the end of the previous run that you reset on.

  5. Please submit runs with audible in-game sound or else they will possibly be rejected.

Be sure to check the category-specific rules for whatever you plan on running, especially for any% where the rules are most strict.
